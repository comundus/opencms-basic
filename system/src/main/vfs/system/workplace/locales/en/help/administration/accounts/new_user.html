In order to create or edit a user within OpenCms all required fields appear as visible input fields:

  1. Login name
  2. Last name
  3. Email
  4. Password
  5. Password Confirmation

are required fields. Optional information does not show an input box but offers a separate "plus" button that allows to add an input field for the information.

Important

If a new user is added it has to be assigned to a group before he / she will be able to login to OpenCms because the administration view grants access to groups. If a single user has no group no login will be possible.
