f804c17a-11b7-11db-91cd-fdbae480bac9 f6df7af7-11b7-11db-91cd-fdbae480bac9 Wenn Sie mit mehreren Redakteuren zusammenarbeiten, ist es nahezu unmöglich zu erkennen wer wann welche Änderungen an einer Datei oder Verzeichnis vorgenommen hat. Dies ist aber für einen reibungslosen Arbeitsablauf unerlässlich denn nur so ist gewährleistet, dass noch nicht für eine Veröffentlichung vorgesehene Dateien verfrüht publiziert werden.

In OpenCms wird daher der jeweilige Status einer Datei oder Verzeichnisses visuell im Workplace dargestellt.

Farben

  • Blaue Beschriftung – Status „Neu“   
    Dateien und Verzeichnisse werden blau dargestellt, wenn diese neu erstellt oder frisch auf den Server hochgeladen wurden. Diese Ressourcen sind noch nicht für den Leser im Internet abrufbar.    
  • Rote Beschriftung – Status „Geändert“         
    Geänderte, in einer früheren Version Online verfügbare Ressourcen werden rot dargestellt. Der Leser im Internet hat auf die aktualisierten Inhalte noch keinen Zugriff, sondern bekommt die vorherige Version angezeigt.
  • Schwarze Beschriftung –  Status „Unverändert“     
    Dateien und Verzeichnisse werden schwarz dargestellt, wenn diese „Online“ und „Offline“ identisch sind.
 

Beachten Sie: Geänderte Dateien sind nicht "Online" (im Internet) abrufbar und entsprechend mit einer roten Flagge gekennzeichnet. Mehr zum Thema "Veröffentlichen" von Dateien finden Sie hier.
