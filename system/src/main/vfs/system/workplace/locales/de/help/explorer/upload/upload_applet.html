Sie können mit Hilfe einer intuitiven Benutzeroberfläche eine oder mehrere Dateien gleichzeitig, oder sogar ganze Verzeichnisse auf den Server hochladen. Ein Geschwindigkeitsvorteil beim Datentransport wird dadurch erreicht, dass alle Dateien (oder die kompletten Verzeichnisse) automatisch in ein komprimiertes Paket (zip) überführt werden und auf dem Server danach wieder entpackt werden.

Der Einsatz lohnt sich vor allem dann, wenn Sie:

  • über einen schnellen Internetzugang verfügen, z.B. DSL, ADSL oder LAN
  • öfters viele Dateien, Dokumente oder Bilder auf den Server hochladen müssen

Nachteil:

  • etwas längere Ladezeit der Benutzeroberfläche.
  • Es wird eine kostenlose Zusatzsoftware benötigt: Die „Java Virtual Machine“ der Firma Sun.
  • Der Ort auf Ihrer Festplatte, von dem aus Sie hochladen wird nicht gespeichert. Wenn Sie erneut hochladen, müssen Sie in der Oberfläche erneut an diesen Ort navigieren. Dies kann bei inkrementellem Arbeiten aufwändig sein, wenn Ihre Bilddateien unter einem langen Pfad lokal auf der Festplatte liegen: Sie erstellen z.B. je nach aktuellem Gebrauch neue Bilder für die Seite die sie gerade bearbeiten, laden das einzelne Bild hoch und verlinken es.

Bitte beachten Sie: Sollte nach dem Aufruf der Option „Neue Datei Hochladen“ lediglich ein graues Rechteck im Inhaltsbereich des Workplace zu sehen sein, ist die benötigte Software nicht auf Ihrem Rechner installiert. Wenden Sie sich in diesem Fall an den Systemadministrator. Wenn Sie auf Ihrem Computer über das Recht eines Computer-Administrators verfügen, dann laden Sie die Java VM kostenlos aus dem Internet und installieren auf sie die Software auf Ihren Rechner. Eine schnelle Internetverbindung wird empfohlen. Sie finden die Software unter der Adresse:  http://www.java.com/de/download/manual.jsp

Bedienung

  1. Navigieren Sie im OpenCms Explorer an den Ort, an dem die Dateien gespeichert werden sollen.
  2. Nach dem Anklicken der Schaltfläche zum Hochladen wird das Applet geladen. Dies kann einige Sekunden dauern.
  3. Wählen Sie das gewünchte Laufwerk im Auswahlmenü aus (siehe Abbildung 1) .
  4. Navigieren Sie (unter Windows durch Doppelklicks, unter Linux durch einmal Klicken) im Dateifenster zu dem Ort, an dem die hochzuladenden Dateien liegen (siehe Abbildung 2). 
  5. Drücken Sie die Schaltfläche OK. Das Hochladen beginnt.
  6. Kontrollieren Sie im OpenCms Explorer, ob die Dateien an der Richtigen Stelle angekommen sind.

Abbildung 1: Auswahl des Laufwerkes.

Abbildung 2: Auswahl von drei Verzeichnissen zum Hochladen mittels gedrückter "Shift" - Taste.


Tip: Es ist möglich, mehrere Dateien gleichzeitig auf den Server zu übertragen und sich so u.U. viel Arbeit zu ersparen. Ähnlich wie im Windows Dateimanager steht Ihnen auch in OpenCms die „Shift“ (Umschalttaste) und „Strg“ Taste zur Verfügung, um eine erweiterte Auswahl treffen. 

Bitte beachten Sie: Bereits vorhandene gleichnamige Dateien werden ohne Nachfrage überschrieben und gehen verloren.
