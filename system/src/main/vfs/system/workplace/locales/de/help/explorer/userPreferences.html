Alle Änderungen, die Sie vornehmen werden von OpenCms gespeichert und bleiben auch beim nächsten Anmelden gültig. Viele Einstellungen erlauben es Ihnen, komfortabler und schneller zu arbeiten. Wenn Sie beispielsweise für längere Zeit an nur einem Projekt arbeiten, und dort hauptsächlich in einem Verzeichnis einer bestimmten Site, dann können Sie OpenCms so konfigurieren, dass sie nach der Anmeldung sofort mit der Arbeit beginnen können: In der richtigen Site, dem richtigen Projekt am richtigen Pfad im Explorer. Ebenfalls können Sie ihre Sprache auswählen und Ihr Passwort ändern.

Über diese Schaltfläche erreichen sie Ihre benutzerspezifischen Einstellungen.

Das Layout ist durch "Karteikarten mit Reitern"  - auch "Tabbed Pane" genannt - realisiert. Sie klicken auf einen Tab und die zugehörige Seite mit den Einstellungen wird sichtbar.

"Workplace"-Tab

  • Sprache:
    Hier können Sie Ihre Sprache einstellen. OpenCms wurde so konzipiert, dass sich nachträglich neue Sprachpakete auch ohne Programmierkenntnisse einfach erstellen lassen. Diese können dann als Module in OpenCms installiert werden. In der Grundversion "spricht" OpenCms Englisch und bietet die Möglichkeit zur Installation eines deutschen Sprachmoduls. Somit lassen sich hier vorerst nur "en" und "de" (die in ISO 639 definierten Sprachcodes) auswählen.
  • Starte mit Projekt:  
    Hier können Sie eines der Projekte, bei denen Sie mitarbeiten, auswählen. Nach jeder Anmeldung in OpenCms befinden Sie sich in diesem Projekt.
  • Starte mit Ansicht :
    Hier können Sie auswählen, in welcher Ansicht Sie nach dem Anmelden an OpenCms beginnen wollen, meist werden sie den Explorer auswählen:
  • Starte in Site :
     Hier wählen Sie die Site, in der Sie sich nach dem Anmelden befinden.
  • Starte in Ordner :
    Hier wählen Sie den Ordner, in dem Sie sich befinden werden, wenn Sie sich anmelden und die Ansicht "Explorer" als Starteinstellung gewählt haben. 
  • Workplace Buttons : 
    Je nach Ihrem persönlichen Geschmack können Sie festlegen, ob die Schaltflächen der Titelleiste als Bild, nur Text oder Bild und Text erscheint.
  • Report Anzeige :
    Hier können Sie einstellen, ob Reporte einfach oder ausführlich dargestellt werden. Immer, wenn sie ein Modul importieren oder exportieren, Links überprüfen lassen, OpenCms installieren, ... sehen Sie Html Reporte mit - per Standard - ausführlichen Informationen über den Vorgang, der gerade ausgeführt wird.
  • Applet zum Hochladen Verwenden :
    Wählen Sie aus, auf welche Weise Sie Dateien auf den Server laden wollen. Mehr Details über die Unterschiede erfahren Sie hier.
