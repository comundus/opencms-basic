Um einen Benutzer für OpenCms zu erstellen oder zu bearbeiten stehen Ihnen für alle erforderlichen Eingaben als Eingabefelder zur Verfügung:

  • Loginname
    Dieser Name muss eindeutig sein: Es darf kein bisheriger Benutzer diesen Loginnamen verwenden. Er wird später beim Anmelden an OpenCms als Eingabe benötigt.
  • Nachname
  • Vorname
  • Email
    Die Emailadresse wird beim Versenden von Nachrichten oder für andere Zwecke benötitg.
  • Passwort:
    Dieses Feld muss mit einem Passowort länger 4 Buchstaben gefüllt sein, wenn das "Aktiviert" Häkchen darüber aktiviert ist.
  • Passwort Bestätigung:
    Dieses Feld muss die exakte Wiederholung des o.a. Passwort Feldes enthalten, um sicher zu gehen, dass das Passwort bekannt ist.

Alle restlichen Felder sind optional:

  • Beschreibung:
    Hier können sie weitere Informationen über die Bedeutung des Benutzers angeben.
  • Adresse
  • PLZ
  • Wohnort
  • Land
