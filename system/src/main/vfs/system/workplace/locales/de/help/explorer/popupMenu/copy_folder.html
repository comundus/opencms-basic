Sie haben beim Kopieren von Verzeichnissen drei Möglichkeiten.


Drei Auswahlmöglichkeiten beim Kopieren von Verzeichnissen.

1.  "Kopiere keine Ressource, erstelle nur Verknüpfungen zu allen Ressourcen im Verzeichnis"

Hierbei wird die Verzeichnisstruktur (alle Unterverzeichnisse) des zu kopierenden Verzeichnisses in ein neues Verzeichnis kopiert. Alle Dateien werden als Verknüpfung kopiert. Eine sinnvolle Aktion, wenn sie eine ganze Site in einer anderen Sprache anbieten wollen.

2.  "Kopiere Resourcen die nicht verknüpft sind, erstelle neue Verknüpfungen für verknüpfte Resourcen"

Hierbei wird die Verzeichnisstruktur (alle Unterverzeichnisse) des zu kopierenden Verzeichnisses in ein neues Verzeichnis kopiert.  Alle enthaltenen Dateien die keine weiteren Verknüpfungen (Geschwister) haben werden inhaltlich kopiert. Für alle Dateien, die mehrere Verknüpfungen haben werden in das neue Verzeichnis kopiert.

Mit dieser Option können Sie verhindern, dass sie Inhalte von Verknüpfungen (mehrfach referenzierte Inhalte) abkoppeln. Vielleicht haben Sie in einem Verzeichnisbaum einige Dateien deren Inhalt einzigartig bleiben muss und bei Änderung überall geändert werden muss (z.B. eine Lizenzdatei). Durch dieses Kopieren bleiben diese Verknüpfungen bestehen, die übrigen Dateien können Sie danach unabhängig von den Orginalen inhaltlich verändern.

3.  "Kopiere alle Resourcen, auch verknüpfte Resourcen"

Kopiert die Verzeichnisstruktur des zu kopierenden Verzeichnisses in ein neues Verzeichnis. Nicht nur die individuellen Dateien werden inhaltlich kopiert, sondern auch die Verknüpfungen ebenfalls. Dies bedeutet bei Verknüpfungen, dass zwar im neuen Verzeichnis auch Verknüpfungen erscheinen werden, die analog zu den orginalen Verknüpfungen auf den gleichen Inhalt zeigen. Jedoch ist dieser Inhalt nur der kopierte Inhalt der orginalen Verknüpfungen. Wenn sie im kopierten Verzeichnis Änderungen an den Inhalten dieser kopierten Verknüpfungen machen, so bleibt der Inhalt der ursprünglichen Verknüpfungen unberührt, lediglich die Inhalte aller Verknüpfungen der Kopie werden manipuliert.

 

Die Auswahl des Zielverzeichnisses geschieht analog zum Kopieren von Dateien .
