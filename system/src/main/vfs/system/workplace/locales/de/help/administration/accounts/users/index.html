In der oberen Gruppe "Benutzer Aktionen" können sie einen neuen Benutzer anlegen.

Die untere Gruppe "Benutzerkonten" gibt Ihnen die Übersicht über die in OpenCms konfigurierten Benutzer und verschiedene Aktionen. Die Darstellung basiert auf einer in der OpenCms 6.0.0 Workplace immer wieder verwendeten Komponente: dem Listen Widget, dessen Bedienung hier erklärt ist.  

Einzelaktionen

Einzelaktionen werden durch Schaltflächen direkt in der Liste gestartet. Sie werden von links nach rechts aufgezählt:

  • Benutzer bearbeiten: Erlaubt nachträgliche Änderungen der Einstellungen des Benutzers.
  • Benutzer zu Gruppen zuordnen: Diese wichtige Aktion, die dem Benutzer letzendlich den Zugriff auf Dateien oder Aktionen ermöglicht ist hier erläutert.
  • Deaktivieren / Aktivieren: Wenn der zugehörige Benutzer aktiv ist, kann er sich in der Workplace anmelden und arbeiten. In diesem Falle ist das Häkchen blau und wird den Benutzer durch Knopfdruck deaktivieren. Wenn der Benutzer bereits inaktiv ist, erscheint hier ein graues Häkchen welches durch Anklicken den Benutzer aktivieren wird. 
  • Löschen: Der Benutzer wird aus der OpenCms Konfiguration entfernt. Mit den alten Benutzerdaten kann man sich danach nicht mehr in der Workplace anmelden.
  • Benutzerübersicht Anzeigen: Diese Aktion wird durch Anklicken des Benutzernamens erreicht.

 

Multi - Aktionen

Multi - Aktionen wirken sich auf mehrere Benutzer aus. Namentlich auf alle Benutzer, die in der letzten Spalte ein Häkchen gesetzt haben. Die Aktionen selbst werden über die Schaltflächen am rechten oberen Ende des Listen - Widgets gestartet, haben aber die gleiche Bedeutung wie die o. a. Einzelaktionen.
