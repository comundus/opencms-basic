  • Ein neues Projekt anlegen
  • Die Standardeinstellungen verändern
  • Die Projekt Historie anzeigen

Die untere Gruppe "Projekte" gibt die Übersicht über die im System vorhandenen Projekte und erlaubt ebenfalls Interaktionen die mit diesen Projekten verbunden sind. Die Darstellung wird von einem sog. Listen - Widget übernommen, dessen Bedienkonzept hier erläutert ist.

Es gibt eine Reihe von Aktionen, die auf die per Häckchen selektieren Projekte ausgeführt werden kann:

  • Projekte Löschen
  • Sperrung Aufheben
  • Pfade der Projekte Anzeigen

Diese Aktionen werden auf alle selektierten Projekte ausgeführt (Multi-Actions).

Weitere Aktionen lassen sich aus einer Zeile heraus, die für ein Projekt steht starten (von links nach rechts):

  • Ansicht der Projektdateien
  • Sperrung  aufheben (wenn eine Sperrung existiert)
  • Projekt veröffentlichen
  • Projekt bearbeiten
  • Projekt löschen

Zum grundsätzlichen Verständnis des Einsatzes von Projekten wird empfohlen, diese Hilfeseite zu lesen. Der Mechanismus und Sinn der Veröffentlichung wird hier beschrieben.
