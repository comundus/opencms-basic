Benennen Sie eine Datei oder Verzeichnis mit Hilfe dieses Menüpunkts um und / oder verschieben Sie diese in ein anderes Verzeichnis. Wenn Sie ein Element verschieben möchten, können Sie nach durch einen Klick auf den „ Suchen “-Button das Zielverzeichnis mit Hilfe eines in einem Extrafenster dargestellten Dateibaums auswählen. Wenn Sie so ein Verzeichnis wählen, wird der Name der Datei nicht verändert, sondern die gewählte Datei in das Zielverzeichnis verschoben.

Möchten Sie stattdessen eine Ressource umbenennen, geben sie einfach den neuen Namen in das Eingabefeld ein und bestätigen mit „ OK “. 

Hinweis: Der Name der unbenannten oder verschobenen Seite oder Datei ist nach der Aktion durchgestrichen  und bleibt bis zur Veröffentlichung erhalten. Diese „durchgestrichene Seite“ oder „ durchgestrichene Datei“ kann jederzeit wiederhergestellt werden. Erst nach dem Veröffentlichen  wird die durchgestrichene Datei entgültig gelöscht. 

Hinweis: Die Namen von Ressourcen dürfen keine Leer- und Sonderzeichen enthalten.

Tip: Zum gleichzeitigen Verschieben und Umbenennen können sie mit der o.g. Suchfunktion ein Zielverzeichnis auswählen und anschliessend in der Textbox noch einen neuen Dateinamen eingeben.
