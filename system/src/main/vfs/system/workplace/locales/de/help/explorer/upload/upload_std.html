Sie können auch ohne zusätzliche Software Dateien in OpenCms hochladen.

Dies ist besonders dann sinnvoll, wenn Sie:

  • nur gelegentlich einige wenige Dateien auf den Server übertragen
  • über eine langsame Internetverbindung (Modem, ISDN) verfügen

Möchten Sie mehrere Dateien auf einmal auf den Server hochladen, müssen diese zuvor als Zip-Datei zusammengefasst werden. Im Dialogfenster „Datei hochladen“ müssen Sie dann die zusätzliche Option “Zip-File entpacken“ das Häkchen setzen.

Nachteile:

  • Langsamerer Datentransport
  • Nur eine Datei pro Upload (oder mehrere Dateien vorher zu einem zip - archiv zusammenfassen)

Vorteile:

  • Keine zusätzliche Software muss installiert werden.
  • Ihr Browser merkt sich, von welchem Ort auf Ihrer Festplatte aus Sie das letzte mal hochgeladen haben. Wenn Sie also häufig einzelne Dateien hochladen, sparen Sie sich Zeit durch das Suchen Ihrer Dateien zum Hochladen.

Diese Variante bietet weit weniger Komfort als die zuvor beschriebene, ist aber für das „gelegentliche“ Hochladen von Dateien auf den Webserver deutlich schneller und völlig ausreichend.

Tip:Bei Windows XP können Sie Dateien ohne Zusatzsoftware als Zip-Archiv zusammenfassen. Markieren Sie die gewünschten Dateien und klicken anschließend mit der rechten Maustaste auf die Auswahl. Im Kontextmenü wählen Sie nun "Senden an Zip-komprimierten Ordner".

Bei älteren Windows-Versionen (98, ME oder 2000) ist eine Zusatzsoftware nötig. Diese können Sie kostenlos unter www.winzip.de herunterladen.
