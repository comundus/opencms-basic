Das oberste Feld "Datum aktualisieren" setzt den "Geändert am" Zeitstempel der Ressource, welches man im Explorer sehen kann. Sie können damit z.B. eine erneute Veröffentlichung erzwingen.

Mit den unteren Eingabefeldern definieren Sie ein Zeitfenster, in dem die Ressource online erreichbar ist. Geben Sie ein Anfangs- und Enddatum für das entsprechende Element ein. Sie können das jeweilige Datum manuell eingeben, wobei Sie das Eingabeschema beachten müssen.

Alternativ steht eine Kalenderfunktion bereit, die Ihnen eine manuelle Eingabe abnimmt. Klicken Sie hierzu auf der rechten Seite des Eingabefeldes auf das jeweilige Kalendersymbol und wählen mit der Maus das gewünschte Datum und Uhrzeit aus. Bestätigen Sie anschließend das Dialogfeld mit "OK". Der Datei- oder Verzeichnisname erscheint nun im Workplace in kursiver Schreibweise. Anhand dieses Merkmals sind mit Zeitfenster versehene Elemente immer gut erkennbar.

Bitte beachten Sie: Unabhängig von der Gültigkeit der Ressource können oder sollten Sie dieseverröffentlichen, damit bei Eintritt der Gültigkeit in der Zukunft auch die aktuelle Version online sichtbar ist. Das Veröffentlichen beeinträchtigt die Online - Erreichbarkeit der Ressource nicht.

Hinweis: Wenn sich eine Ressource ausserhalb Ihrer sichtbaren Zeit im Explorer betrachten, erscheint sie kursiv.
