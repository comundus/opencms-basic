f756a6cf-11b7-11db-91cd-fdbae480bac9 Mit diesem Menüpunkt erstellen Sie eine normale Inhaltsseite, die Ihre Inhalte in dem von Ihnen gewählten Template-Layout  darstellt.  Nachdem Sie im Dialogfenster „Neu“ die Auswahl „Seite“ gewählt haben, wird der Dialog entsprechend fortgeführt. In zunächst zwei weiteren Dialogfenstern werden alle relevanten Werte abgefragt.

  • Name       
    Vergeben Sie einen eindeutigen und verständlichen Namen für die neue Datei. Beachten Sie, dass der Dateiname weder Leerzeichen, noch Sonderzeichen (ä, ü, ö, ß....) enthält.
  • Template
    Auswahl des zu verwendenden Templates. Der Wert ist Projektspezifisch und müsste Ihnen mitgeteilt worden sein.
  • Kopiere Body von         
    Anzahl der im Inhaltsbereich des Templates angezeigten Elemente. Die Anzahl kann nachträglich im Editor geändert werden.
  • Sofern die Eigenschaft „Bearbeite Eigenschaften der neuen Datei“ aktiviert wurde, werden in einem Folgedialog das Editieren der Eigenschaften (basics und "erweiterte") angeboten.
  • Dialogfenster: Eigenschaften  
    Füllen Sie in diesem Dialogfenster alle wichtigen Eingabefelder aus. Bestätigen Sie anschließend mit einem Klick auf „ Fertig stellen“.
