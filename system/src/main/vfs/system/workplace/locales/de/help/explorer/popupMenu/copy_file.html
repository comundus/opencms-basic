  1. Sie komplexe Verzeichnisstrukturen duplizieren wollen.
  2. Sie wiederkehrende Layouts oder Merkmale in Dateien übernehmen wollen.
  3. Sie viele Eigenschaften ("properties") an Ressourcen gehängt haben und diese auf neue Ressourcem übernehmen wollen. 
  4. Sie eine Datei mit gleichem Inhalt, jedoch von einer anderen Stelle in Ihrer Navigation aus (und womöglich mit einem anderen Namen und weiteren geänderten Eigenschaften) benötigen.
  5. Sie die gleiche Datei in einer anderen Sprache zu Verfügung stellen wollen.


Kopieren als neue Ressource oder als Verknüpfung.

Kopieren als neue Ressource

Diese Option müssten Sie im Kopierdialog für die ersten drei der oben beschriebenen Aufgaben wählen. Hierbei wird die ausgewählte Ressource wirklich kopiert - d.h. ihr Inhalt wird gelesen und in neue Dateien geschrieben.

Erstellen einer neuen Verknüpfung zur Datei

Diese Option müssten Sie wählen, wenn sie die letzten beiden der oben beschriebenen Aktionen ausführen. Der Begriff Kopieren ist hier etwas missverständlich: Zwar sehen Sie als Resultat im Explorer wirklich eine zweite Datei, die jedoch den gleichen Inhalt wie die ursprüngliche Datei hat. Beide Dateien sind gleichwertige Verknüpfungen ("Geschwister"): Jedesmal, wenn Sie den Inhalt einer Datei editieren, so ist der Inhalt aller Verknüfungen auch geändert.

Verknüpfungen eignen sich dazu, (im Explorer) andere Eigenschaften an gleiche Inhalte zu setzen oder gleiche Inhalte unterschiedlich zu benennen. Sie können z.B. über eine Eigenschaft "language" die Sprache der Verknüpfung kontrollieren und innerhalb ein und derselben Datei (im Inhalt) beide Sprachen zur Verfügung stellen.

Tip: Mehr über Verknüpfungen erfahren Sie hier.

Kopieren nach

Hier geben Sie den Ort im Dateisystem für die neue Verknüfung ein.

  • Wenn sie die Datei einfach im gleichen Verzeichnis kopieren wollen, geben sie einen Namen ein.
  • Wenn sie die Datei in einem anderen Verzeichnis erstellen wollen, klicken Sie auf die rechtsseitige Schaltfläche, die ein Verzeichnis darstellt. Der VFS - Explorer erlaubt Ihnen eine Auswahl des Zielverzeichnisses. Nach Bestätigung können Sie nun:
    • Auf "OK" klicken: Die Datei wird unter gleichem Namen in das andere Verzeichnis kopiert.
    • Einen Dateinamen an den im Textfeld erschienenen Pfadnamen anhängen und dann auf "OK" klicken: Die Datei wird unter neuem Namen in das Neue Verzeichnis kopiert.
